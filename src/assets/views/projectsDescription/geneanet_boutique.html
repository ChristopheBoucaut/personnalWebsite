Développement de la nouvelle boutique de Geneanet avec Prestashop.

La boutique se comporte comme une gallerie marchande. Les partenaires de Geneanet proposent leurs produits via cette plateforme. Dans ce but, une surcharge propre a été effectuée. On peut ainsi suivre l'état de la commande pour chaque vendeurs concernés. Un intranet a également été développé pour permettre aux partenaires de gérer leur boutique. Cet intranet exploite les API de Prestashop.
