Développement d'une application web pour gérer ses vaisseaux dans le jeu Elite Dangerous.

Pour un besoin personnel, j'ai développé une application me permettant de gérer mes vaisseaux dans le jeu en ligne Elite Dangerous. Il m'est ainsi possible de suivre mes vaisseaux, leur position, le prix, lister les choses à faire dessus et d'ajouter des notes (ex : vaisseau pour le commerce, vaisseau pour le combat, etc).
