Développement d'une progressive web app.

Pour un besoin personnel, j'ai développé une PWA me permettant de gérer des tâches périodiques dont la prochaine itération doit démarrer depuis la dernière fois que la tâche a été effectuée. L'avantage d'une PWA dans ce projet est d'avoir une pseudo application android sans devoir remettre en place Cordova pour un petit projet.
